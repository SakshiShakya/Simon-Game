The Simon Game

Rule-Book

Press any key. Simon will give the first signal. Repeat the signal by pressing the same color button.

Simon will add one more signal. Repeat these two signals by pressing the same color buttons, in order.

Simon will add one more signal. and on and on!

Continue playing as long as you can repeat each sequence of sig-nals correctly.

If you fail to repeat a sequence exactly. you lose!
